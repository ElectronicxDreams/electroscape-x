    Audio Sequence Protocol

    01 — FAVOURITES: SAVING TRACKS

    Click the HEART ICON (♥) on any track card to save it. The icon turns solid cyan when a track is favourited. Your list is stored locally and persists across sessions.

    02 — SIDEBAR: MANAGING LISTS

    Open the FAVOURITES sidebar via the action bar heart button. From here you can:

    • PLAY ALL: Start the full favourites queue.
    • REORDER: Use the drag handle (6 dots) or arrows (▲/▼).
    • REMOVE: Click ✕ on any row to delete it.

    03 — QUEUE: EXITING FAV MODE

    While in Favourite Mode, SELECT ANY TRACK from the main grid to exit the favourites queue and restore all tracks.

    04 — NAVIGATION & VIEW

    Use OVERRIDE SEQUENCE to shuffle the grid. Use the GRID ICONS (Micro, Core, Max) to switch display density. The ORBITAL HUD at the bottom provides persistent controls and seek.
